# Resume Bullets

All bullets describe a portfolio project built on synthetic data. No real employer or client impact is claimed.

## Technology Risk / IT Audit

- Built a synthetic-data access-control exception testing project in SQL/SQLite, testing 500 users and 1,000 access requests against deprovisioning, MFA, and approval controls, identifying 27 terminated-user access exceptions and 28 privileged users without MFA.
- Designed and documented nine SQL-based control tests (access, approval-timing, business-justification, and IT ticket workflow) with plain-language risk, control, and exception logic, and summarized results in a risk-rated control matrix and executive memo.

## Business Systems Analyst

- Analyzed 3,000 synthetic IT service desk tickets in SQL to identify the highest SLA-breach (23.42%), reopen (23.08%), and escalation (36.71%) categories, translating workflow data into process-improvement recommendations.
- Built a Power BI dashboard MVP (KPI cards and a findings/control matrix table) from SQL query outputs, including DAX measures with robust text/boolean handling for flag-based data fields.

## Risk / Operations Analyst

- Created a synthetic access-request dataset and SQL testing framework to quantify control exceptions (e.g., 84 instances of access provisioned before approval, 143 instances of missing business justification), and mapped each exception to a documented recommendation.
- Produced audit-style deliverables — data dictionary, control matrix, executive summary, and interview walkthrough — for a synthetic IT controls and workflow risk portfolio project, demonstrating end-to-end analyst workflow from raw data to leadership-ready reporting.
